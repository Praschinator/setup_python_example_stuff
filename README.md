## Development

to create venv run: 
```
python3 -m venv lss_automated_remastered-venv
```


to activate venv run:
linux: 
```bash
source lss_automated_remastered-venv/Scripts/activate
```
windows: 
```powershell
.\lss_automated_remastered-venv\Scripts\Activate.ps1
```


then: 
```
pip install -r reqirements.txt
```
